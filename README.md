# Student Gradebook App

A console-based C# application that manages student records and grades while demonstrating the use of common collection types.

## Features

* Add and manage students
* Assign and store grades
* View and search student records
* Calculate average grades
* Track unique course codes
* Manage an assessment queue
* Record recent actions using a stack

## Collections Used

* **List** - Store students and grades
* **Dictionary** - Map students to subjects/grades
* **HashSet** - Store unique course codes
* **Queue** - Manage students waiting for assessment
* **Stack** - Track recent actions

## Learning Objectives

* Work with collections in C#
* Add, update, remove and search data
* Iterate and sort collection items
* Select appropriate collection types for different scenarios
